# Don Works

Open-source by Revitt.

This repository contains the Next.js landing page for `donworks.co.uk` — where we open up the tools we build at Revitt, starting with [MCPlexer](https://github.com/Don-Works/mcplexer) and [Brw](https://github.com/Don-Works/brw).

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
